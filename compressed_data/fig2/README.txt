This folder contains the raw data for the spectrum plotted in figure 2.

All data file names have the object name and the instrument name.  As there are two LRIS spectra for AT2024qfm, 'AT2024qfm_lris_new.spec' refers to the spectrum that was taken later chronologically.  All data files are tab-separated text files, except for AT2024aehp_GMOS.txt, which is a comma-separated text file.  The wavelength column name is 'wavelength', and the flux column name is 'flux'.

For each file, I will proceed to describe whether it provides the 1-sigma uncertainty or the variance (1-sigma uncertainty squared), that column's name and the zero-indexed header row location.

AT2022abfc_GMOS_stack.txt has the header on row zero, and has a variance column named 'unc'.

AT2023fhn_p200.txt has the header on row zero, and has a 1-sigma uncertainty column named 'fluxe'.

AT2023fhn_GMOS.txt has the header on row zero, and has a 1-sigma uncertainty column named 'fluxe'.

AT2023fhn_LRIS.txt has the header on row 166, and has a 1-sigma uncertainty column named 'unc'.

AT2023hkw_Keck.txt has the header on row zero, and has a 1-sigma uncertainty column named 'unc'.

AT2023vth_GMOS.txt has the header on row zero, and has a variance column named 'unc'.

AT2023vth_SEDM.txt has the header on row 210, and has a variance column named 'unc'.

AT2024qfm_lris.spec has the header on row zero, and has a 1-sigma uncertainty column named 'unc'.

AT2024qfm_lris_new.spec has the header on row zero, and has a variance column named 'unc'.

AT2024qfm_GMOS.tsv has the header on row zero, and has a variance column named 'unc'.

AT2024qfm_binospec.ascii has the header on row 46, and has a 1-sigma uncertainty column named 'unc'.

AT2024aehp_GMOS.txt has the header on row zero, and has a variance column named 'unc'.

AT2024aehp_KCWI.txt has the header on row zero, and has a 1-sigma uncertainty column named 'err'.

AT2024aehp_binospec.txt has the header on row zero, and has a 1-sigma uncertainty column named 'unc'.
